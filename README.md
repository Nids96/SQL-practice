# SQL Practice

My journey learning SQL from scratch, with the goal of becoming job-ready as a data analyst.

This repo collects the queries I write as I learn — saved day by day so I can see my own progress over time.

## What I'm following

- **[SQL Full Course for Beginners](https://www.youtube.com/watch?v=SSKVgrwhzus)** by Data with Baraa — my main course for learning the language.
- A **90-day Data Analyst roadmap** — practice problems (LeetCode, DataLemur) and milestone projects that I drill once I've learned a topic.

## Tools

- **PostgreSQL** (via [Supabase](https://supabase.com)) — where I write and test every query, right in the browser.

## How this repo is organized

Each day of practice gets its own `.sql` file, grouped into weekly folders:

```
week-01/
  day-01-basics.sql
  day-02-intermediate.sql
week-02/
  ...
```

Every file starts with a comment noting the date and what it covers.
